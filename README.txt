Gagstock Tracker
===============

This is the magical, animated, aesthetic Grow a Garden stock tracker website.

To run locally:
- Open index.html in a modern browser.

This project includes:
- Animated particles background
- Loading screen with animated icon and gradient text
- Theme switcher (3 themes)
- Circular countdown timers and progress bars
- Animated weather Lottie icon
- PWA manifest and service worker for offline caching

Created by Sunnel.
